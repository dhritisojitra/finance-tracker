Key Features:
- 📊 Interactive Dashboard UI: Real-time display of financial summaries using dynamic state
- 📈 Data Visualization: Charts for expense categories and income analysis
- 💳 Transaction Interface: Clean UI for viewing and managing transactions
- 🎯 Savings Goals UI: Goal tracking with visual progress
- 🧾 Invoice UI (Download Ready): Frontend implementation for generating transaction records
- 🧠 Finance Learning Section: Beginner-friendly investing tips and guidance
- 🧩 Customizable Layout: Widget-based dashboard for personalized experience

🛠️ Tech Stack:
- React (Vite)
- Tailwind CSS
- Framer Motion
- JsPDF

🧠 Approach:
- Built using component-based architecture for reusability
- Focused on UI/UX clarity and responsiveness
- Used state management (React hooks) to handle dynamic data
- Designed modular dashboard cards for scalability
- Simulated real-world features like admin controls and transactions

⚙️ Setup Instructions:

1. Clone the repo
git clone https://github.com/dhritisojitra/wallet.git
cd wallet

2. Install dependencies
npm install

3. Run locally
npm run dev

4. Build for production
npm run build
